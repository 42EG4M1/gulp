# gulp boilerplate
Gulp boilerplate for my basic project.  
(Gulp Starter)


## Require
* Node.js


## Getting Started
### Clone
    $ git clone git@github.com:42EG4M1/gulp-boilerplate.git [projectName]

### Install npm packages
    $ npm i

### Build

    // gulp task
    $ npm run gulp


### Run only when New or Changed

    // svg sprite
    $ npm run svg

    // favicon
    $ npm run favicon
